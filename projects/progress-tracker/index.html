Progress Tracker

  • Expo
  • React Native

Progress tracker is a task management tool that lets users track progress of their tasks (or goals). Each task progress is depicted by the percentage of completed subtasks.

Source

Features

In addition to creating, browsing, editing and deleting tasks, users have the ability to:
  • horizontal_rule Prioritize Tasks: Select the most important tasks and pin them to appear at the top.
  • horizontal_rule Set Deadlines: Make tasks time bound by setting a due date.
  • horizontal_rule Manage Subtask Size: Set subtask weights to represent larger subtasks better.
  • horizontal_rule Group Tasks: Categorize tasks by giving them labels.

Managing subtasks

The progress of a goal (task) depicts the percentage of completed subtasks.

Unless specified otherwise, all subtasks contribute equally to the progress of a goal, as weights default to the value of 1.
