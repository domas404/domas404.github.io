Progress Tracker
Expo
React Native
A task management tool that let's users track progress of their tasks (or goals). Each task progress is depicted by the percentage of completed subtasks.
Source

What can it do?

This app not only let's users create, edit and delete tasks, it also gives users the opportunity to:
horizontal_rule Prioritize tasks by pinning them.
horizontal_rule Make tasks time bound by setting a due date.
horizontal_rule Sort tasks by creation date/due date/alphabet/progress to help manage between many tasks.
horizontal_rule Set weights to subtasks to represent larger subtasks better.
horizontal_rule Categorize tasks by giving them labels.

Subtasks

This app is well suited for school or university assignments, as it let's users note important things by adding description and group tasks by labels.
