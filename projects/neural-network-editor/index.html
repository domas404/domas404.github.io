Neural network editor is an educational tool that allows users to design simple model architectures and train them on selected datasets.
This intended audience of this web application are people who do not have strong STEM background but are interested in learning how neural networks work.

Creating a neural network

Users can build model architecture by dragging and dropping layers. Neuron count and activation function in each layer can be changed in the top right panel.
