Neural network editor

  • Next.js
  • Tailwind CSS
  • TensorFlow.js
  • TypeScript

Neural network editor is an educational tool that allows users to design simple model architectures and train them on selected datasets.

open_in_new Demo
Source

What can it do?

This application lets users interactively build and train shallow neural network architectures using graphical user interface.

Users can:

  • horizontal_rule Create neural network architectures via drag & drop functionality
  • horizontal_rule Change properties of each neural network layer
  • horizontal_rule Select a dataset to use for training
  • horizontal_rule Try various hyperparameter options to see which ones make the model perform best
  • horizontal_rule Navigate between several different models

Choosing data

Users can select a dataset from the provided list of datasets and filter it by features or classes.

Training model

Training computations are performed on the client's side, utilizing user's device resources.

When training is finished, users can view loss and accuracy plots, as well as classification matrix to evaluate model's performance.
