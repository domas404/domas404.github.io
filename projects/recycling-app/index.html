Recycling App

In progress
  • Expo
  • React Native
  • TypeScript

A mobile app designed to help users dispose of waste responsibly by providing clear instructions, maps to nearby waste drop-off points, and custom reminders.

Source

Features

  • horizontal_rule Search for Waste Items: Quickly find the correct way to dispose of various items by browsing categories or entering keywords.
  • horizontal_rule Recycling Best Practices: View do's and don'ts of recycling and other important tips.
  • horizontal_rule Nearby Collection Points: Find the locations of the nearest waste disposal sites.
  • horizontal_rule Waste Pick-Up Reminders: Set periodic reminders for your local waste collection days.

Searching for items

Users can search for various waste items and view package labeling explanations.

Next to each waste item, it is clearly indicated where such an item should be disposed of. Most items have additional information detailing the nuances of recycling.
